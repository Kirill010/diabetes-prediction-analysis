# diabetes-prediction-analysis

- Описание задачи: предсказание наличия диабета по медицинским параметрам (датасет Pima Indians Diabetes).
- Цель: продемонстрировать навыки анализа данных, предобработки, построения и оценки модели классификации.
- Данные: кратко о признаках (количество беременностей, уровень глюкозы, давление, ИМТ и т.д.).

- Этапы работы:

    1. Исследовательский анализ (корреляции, распределения, сравнение групп).

    2. Предобработка (нормализация, train/test split).

    3. Модель: логистическая регрессия.

    4. Оценка: accuracy, confusion matrix, ROC-AUC, precision/recall.

- Результаты:

    1. Accuracy: ~0.71

    2. ROC-AUC: ~0.82

- Выводы: модель позволяет с умеренной точностью предсказывать диабет; для улучшения можно попробовать другие алгоритмы или собрать больше данных.
- Стек технологий: Python, pandas, numpy, matplotlib, seaborn, scikit-learn.
- Ссылка на Kaggle (https://www.kaggle.com/datasets/mathchi/diabetes-data-set/data).
